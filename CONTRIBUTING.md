# Contributing

Thanks for helping improve this project.

## Prerequisites
- Node.js (LTS)
- npm

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run build` (or `npm run watch` while developing)

## Branching
- Create a feature branch from `main`.
- Use short, descriptive branch names. Example: `feature/add-login-route`.

## Commits
- Write clear, present-tense messages.
- Example: `Add BM agent status page`

## Pull Requests
- Keep PRs focused and small.
- Include a brief summary and testing notes.
- Request review from at least one teammate.

## Code Style
- Follow existing patterns in the cartridge.
- If you change lint or formatting rules, call it out in the PR.
